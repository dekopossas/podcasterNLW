# podcasterNLW
## Descrição
Aplicativo de Podcast produzido na quinta edição do Nest Level Week / Rocketseat.

### Conteúdo Trabalhado
- Fundamento Web
  - <b>refs em React</b>
  - Áudios HTML
- Fluxos de API
- React
  - Componente
  - Propriedade
  - Estado
  - limitações
  - ContextAPI
- Next
  - Server-side rendering
  - Static site generation
  - <b>Root file system</b>
  - Método Geting Static Paths
    - Fallback
- <b>Type Script</b>
- CSS modules
- <b>SASS</b>
  - Fontes Externas
- JSON Server
- Axios API
- Diferenças entre SPA SSR e SSG
- HOME estático
- Conceitos de BUILDNG
- <b>Lógica Completa de um Player</b>
  - Play, Pause, Next, Previous
  - Loop, Shuffling
  - Slider Control
